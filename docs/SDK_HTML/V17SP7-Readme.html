Release Notes:
--------------

Product:        Solid Edge
Version:        17.00.07.03
Date:           29-Nov-2005


This release of Solid Edge has been certified to run on the following:
 • Windows 2000 Professional® operating system running Service Pack 4.
 • Windows XP Professional® operating system running Service Pack 1 or 2.
 • Internet Explorer 6.0 or higher


This Update should be installed on top of Solid Edge Version:
17.00.00.65, 17.00.01.06, 17.00.02.04, 17.00.03.04, 17.00.04.03, 17.00.05.03,
17.00.06.03


--------------------
End of Release Notes

------------------------------------
This Update addresses the following:
------------------------------------

=====Assembly=====

PR 2055227: Fixed a problem where Solid Edge exits when opening this assembly - Japanese OS Specific.
PR 5344939: Fixed a problem based on crashlog files that should resolve several aborts.
PR 1452562: Fixed a problem where Edgebar was not updated after revising a component
PR 5331276: Fixed a problem with the Registry setting for Place Part/Assemble to be Mate
PR 5332702: Fixed a problem with file save in assembly
PR 5346171: Fixed a problem with incorrect part texture being loaded when opening assemblies
PR 2055627: Fixed a problem that cased an abort when saving an assembly configuration.
PR 1468239: Fixed a problem activating display a configuration of a subassembly
PR 2055667: Fixed a problem where you can not open Sub assembly from Main assembly
PR 2055673: Fixed a problem where Configuration list is not retained when returning to a subassembly
PR 5348651: Fixed a problem where FOA is not behaving correctly when changing members
PR 5349577: Fixed a problem where Assembly features are not showing graphically in the display.
 
 
=====Part=====

PR 5345416: Fixed a problem that caused Solid edge abort when closing document or exiting the application.
PR 1447589: Fixed a problem that caused an abort when trying to make changes within property manager
PR 5338890: Fixed a problem with features failing with "Unable to recompute topology tracking information"
PR 1466602: Fixed a problem where Changing material name does not fire off recompute for custom formula
PR 1469431: Fixed a problem with Sketch Relationship Color affecting feature update with Dynamic Edit
PR 5348473: Fixed a problem where Material gages are not being saved.

=====Draft=====
 
PR 2036389: Fixed a problem where Sketch shifts using "by 2 points" dimension
PR 2047352: Fixed a problem where All elements disappear after panning
PR 1469482: Fixed a problem where hidden holes in mirrored partcopy display visible

=====API=====

PR 1449660: Fixed a problem where the entire part is green when opening parasolid from V8-V14 files that had threads.
PR 5344764: Fixed a problem where Revision Manager Replace API is not working correctly when fixing links to FOA members

=====Insight=====

PR 1466763: Fixed a problem with Insight API - DeleteDocumentsFromServer
PR 5347635: Fixed a problem where Property Manager does not list any FOP Children when in the master file
