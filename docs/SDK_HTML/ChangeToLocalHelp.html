User Assistance > Change To Local Help

When web help does not open, Solid Edge automatically displays a message containing a check box that lets you set locally-installed help as the default.

You can always set this manually by doing the following:

  1. Click the Application button , and then Settings > Options.

  2. In the Solid Edge Options dialog box, click the Helpers tab.

  3. In the Help System section, check following check box:

    • Access help from my server (requires you to setup a help server).

  4. Click Apply.

    You do not have to exit the program for the change to take effect.

Note:

The next time you access Programming help by clicking the link, local help will be automatically launched from the installed location: C:\Program Files\Siemens\Solid Edge 2019\SDK.
