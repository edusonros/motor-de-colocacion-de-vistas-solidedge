Release Notes:
--------------

Product:        Solid Edge
Version:        17.00.11.03
Date:           20-Apr-2006


This release of Solid Edge has been certified to run on the following:
 • Windows 2000 Professional® operating system running Service Pack 4.
 • Windows XP Professional® operating system running Service Pack 1 or 2.
 • Internet Explorer 6.0 or higher

This Update can be instaled installed on top of Version:
17.00.00.65, 17.00.01.06, 17.00.02.04, 17.00.03.04, 17.00.04.03, 17.00.05.03,
17.00.06.03, 17.00.07.03, 17.00.08.03, 17.00.09.03, or 17.00.10.02.


Note: Updates are all inclusive and can be installed on top of the base release
or any released UPdate.


--------------------
End of Release Notes
--------------------

------------------------------------
This Update addresses the following:
------------------------------------

=====Assembly=====

PR 1511090: Fixed a problem where Solid Edge crashes during assembly file-open
PR 1480472: Fixed a problem with an assembly feature
PR 1509025: Fixed a problem where a step translation is missing sub assemblies on export
PR 1509100: Fixed a problem where you are unable to select a part in a subassembly while in-place active
PR 1512311: Fixed a problem where Mirror assembly fails if the template files are read only
PR 2062106: Fixed a problem where you cannot display sketch of other part when editing a part
PR 2063869: Fixed a problem where Measure minimum distance command does not work properly in IPA'ed sub-asm
PR 2064286: Fixes a problem where Solid Edge crashed in assembly locate.
PR 2065127: Fixed a problem where Solid Edge crashed in the fit command after selecting the "activate" command.
PR 2063373: Fixed a problem where you can't save a "wrl" file with a name or folder containing double byte characters.
PR 2065402: Fixed a problem with the Inactivate All option when opening assembliess
PR 2064492: Fixed a problem when using Edit Definition on a Part and try to apply any other relationship type besides Mate or Planar Align and it requests you select the part again.

=====Part=====

PR 2063942: Fixed a problem with Select step causes an abort editing a pattern
PR 5430418: Fixed a non reproducible crash when exiting profile environment
PR 5414713: Fixed a problem with FOP members Material gets reset to parent material if populate
PR 5446562: Fixed a problem where Solid Edge was producing incomplete crashlogs
PR 5442422: Fixed a problem with SetThePixelDust leaks gdi resources.
PR 5445871: Fixed a problem where Solid Edge was producing incomplete crashlogs when running in 3GB
PR 5446096: Improved crashlog data.

=====Draft=====

PR 1471552: Fixed a problem with FOP variables change from driving to driven.
PR 2063829: Fixed a problem with When the drawing view is updated, the reference element of the dimension is change
PR 2064122: Fixed a problem with If segment is changed with Edge Painter, entire edge changes on update
PR 5360630: Fixed a problem that caused a crash in group dimensions.
PR 5446069: Fixed a problem that caused a crash in group dimensions.
PR 2064302: Fixed a problem where an inserted bitmap object was displaying as dimension color.
 
=====Insight=====

PR 1500061: Fixed a problem with Bad performance when copying with insight a big assembly.
PR 5393910: Fixed a problem with Family of parts stay released when copied.
