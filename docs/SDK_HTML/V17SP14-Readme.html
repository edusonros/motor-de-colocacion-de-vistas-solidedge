Release Notes:
--------------

Product:        Solid Edge
Version:        17.00.14.01
Date:           14-Sep-2006


This release of Solid Edge has been certified to run on the following:
 • Windows 2000 Professional® operating system running Service Pack 4.
 • Windows XP Professional® operating system running Service Pack 1 or 2.
 • Internet Explorer 6.0 or higher

This Update can be instaled installed on top of Version:
17.00.00.65, 17.00.01.06, 17.00.02.04, 17.00.03.04, 17.00.04.03, 17.00.05.03,
17.00.06.03, 17.00.07.03, 17.00.08.03, 17.00.09.03, 17.00.10.02, 17.00.11.03,
17.00.12.01, or 17.00.13.01


Note: Updates are all inclusive and can be installed on top of the base release
or any released UPdate.

End of Release Notes

------------------------------------
This Update addresses the following:
------------------------------------

=====Assembly=====
 
PR 5559905: Fixed a problem that caused a crash in JAsmSmartLocateCmd
PR 5562290: Fixed a problem that caused a crash in JDocOle::Set/GetClientSite
PR 5562502: Added code to prevent crashes while accessing stale pointers.
PR 5564472: Fixed a problem that caused a crash in JOLEMembassy methods
PR 1530114: Fixed a problem where a specific file could not be opened.
PR 5190469: Fixed a problem where Sharpen Display setting #5 (finest setting) is not working correctly
 
=====Part=====
 
PR 5562358: Fixed an abort based on crashlog info.
PR 5569600: Fixed a problem that caused a crash in SignificantEvents::MouseTrackHook
PR 1482704: Fixed a problem where Insert > Partcopy (FOP) causes to insert wrong members
PR 2067805: Fixed a problem where element highlighted incorrectly by Include command.
PR 5561336: Fixed a problem where MRU list created with bad characters in place of http
 
=====Draft=====
 
PR 2065591: Fixed a problem that caused an abort when manipulating objects.
PR 2068664: Fixed a problem that caused an abort when updating hole table.
