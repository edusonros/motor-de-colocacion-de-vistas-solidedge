Release Notes:
--------------

Product:        Solid Edge
Version:        18.00.10.04
Date:           19-Dec-2006


This release of Solid Edge has been certified to run on the following:
    • Windows XP Professional® operating system (32-bit) running Service Pack 2
    • Windows XP Professional® operating system (64-bit) running on EM64T hardware
    • Windows 2000 Professional® operating system running Service Pack 4
    • Internet Explorer 6.0 or higher


This Update can be instaled installed on top of Version: 18.00.00.69, 18.00.01.08,
18.00.02.04, 18.00.03.04, 18.00.04.07, 18.00.05.05, 18.00.06.06, 18.00.07.04,
18.00.08.03 or 18.00.09.06.


Note: Updates are all inclusive and can be installed on top of the base release
or any released UPdate.

------------------------------------
This Update addresses the following:
------------------------------------

=====Assembly=====

PR 1512560: Fixed a problem where pipe report is reporting incorrect cut lengths.
PR 1517589: Fixed a problem where V18 ignores the inactive/active status when changing display configurations
PR 1538648: Fixed a problem where FOA/AP assembly 'hangs' Solid Edge when saving
PR 1554189: Fixed a problem where in the assembly was taking a long time to open in V18 - a regression from V17.
PR 2071892: Fixed a problem where the pipe route is not hidden
PR 2072816: Fixed a problem with transferring assembly relationships from one part to another.
PR 5576244: Fixed a problem with 'zoom to' while editing a subassembly
PR 5606880: Fixed a problem where Interference Check is not honoring occurrence properties for sub-assembly
PR 5532555: Fixed a problem where Subassembly display configurations cause all parts to be hidden.
PR 5613967: Fixed a problem where parts disappear when using Edit in Context of Assembly

=====Part=====

PR 1549673: Fixed a problem where querying custom properties caused SE to crash
PR 1526713: Fixed a problem with the OLE for DM application Wizard
PR 1549719: Fixed a problem where Assembly features are not visible
PR 2070821: Fixed a problem where Feature recompute time is too slow on some files
PR 5610135: Fixed a problem where Undo results are incorrect after making profile changes in Part.

=====Draft=====

PR 5568157: Fixed a problem where Solid Edge crashes after updating hole table/drawing views
PR 5596799: Fixed a problem where a Draft file will not open
PR 1551045: Fixed a problem where Solid Edge hangs when placing a callout with property text
PR 1554426: Fixed a problem where complex strings are displaying with wrong style.
PR 5571394: Fixed a problem where visible lines are not mapping correctly into AutoCAD.
PR 5612274: Fixed a problem with automatic centerlines not placing on three cylinders on this drawing.
PR 5613183: Fixed a problem where shaded views are not printing on larger Draft files.
PR 5373210: Fixed a problem where property text shows an error if the underlying model file is missing.
PR 5618414: Fixed a problem where old DQ views are updating incorrectly.
PR 1557190: Fixed a problem where complex strings are not displaying in block occurrences.
PR 1540562: Fixed a problem with differences in thread lines between V17 and V18.
PR 1554625: Fixed a problem where Draft opens in V16 and V17, but not in V18 or V19

=====API=====

PR 1553018: Fixed a problem with Crash in get_faces method
PR 1554961: Fixed a problem with QueryInterface fails for AssemblyFeatureHole
PR 1555134: Fixed a problem with GetScopeParts fail with a type mismatch error
PR 1538697: Fixed a problem with bend angle in GetBendCenterLineAttributesEx() API

=====Insight=====

PR 1483158: Fixed a problem where RevMan Copy is not executed when "Prevent duplicate file names in Insight" is used
PR 1541078: Fixed a problem where renaming a managed part in Revision Manager does not work.
PR 1551714: Fixed a problem where performance on file open very slow in V18SP8 compared to V16SP9.
PR 1557352: Fixed a problem where ServerMap.txt doesn't work - Insight still tries to access the old server
PR 1557662: Fixed a problem where Life Cycle Assistant doesn't move documents
PR 5648569: Fixed a problem where you cannot open and revise on the fly from internet explorer dashboard
PR 1560300: Fixed a problem where LCA Dialog doesn't update the file list for the SharePoint doclib folder
