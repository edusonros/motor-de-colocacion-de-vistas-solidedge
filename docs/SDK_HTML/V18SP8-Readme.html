Release Notes:
--------------

Product:        Solid Edge
Version:        18.00.08.03
Date:           29-Aug-2006


This release of Solid Edge has been certified to run on the following:
    • Windows XP Professional® operating system (32-bit) running Service Pack 2
    • Windows XP Professional® operating system (64-bit) running on EM64T hardware
    • Windows 2000 Professional® operating system running Service Pack 4
    • Internet Explorer 6.0 or higher


This Update can be instaled installed on top of Version: 18.00.00.69, 18.00.01.08,
18.00.02.04, 18.00.03.04, 18.00.04.07, 18.00.05.05, 18.00.06.06, or 18.00.07.04.


Note: Updates are all inclusive and can be installed on top of the base release
or any released UPdate.

End of Release Notes

------------------------------------
This Update addresses the following:
------------------------------------

=====Assembly=====

PR 5559317: Fixed a problem where deleting a part causes an abort
PR 5555839: Fixed a problem where physical properties of an assembly can not be updated
PR 1533556: Fixed a problem where PartFinder will terminate with SMAP - 3D error
PR 5519203: Fixed a problem with Parasolid based import translation performance
PR 5556450: Enabled Migration commands with Solid Edge license feature

=====Part=====

PR 1527140: Fixed a problem where Solid Edge hangs with a PWD file
PR 1535616: Fixed a problem where a gage does not honor choice after switching tabs
PR 5561336: Fixed a problem where the MRU list has bad characters in place of http
PR 5562891: Fixed a problem where deleting a sick relationship causes Solid Edge to abort.
PR 1538352: Fixed a problem where thread color is not displayed in V17 and V18.
PR 5566826: Fixed a problem with File crashes with thread textures on
PR 5569802: Fixed a problem with IGES performance issue caused by error logging


=====Draft=====

PR 1534055: Fixed a problem where updating a section view causes an abort
PR 5558000: Fixed a problem where opening a file crashes Solid Edge
PR 1516299: Fixed a problem with dimensioning to blocks inside broken views.
PR 1516308: Fixed a problem where exporting a mirrored block is in the wrong orientation
PR 5563211: Fixed a problem where blocks nested in a symbol do not export to AutoCAD
PR 1536329: Fixed a problem with smart hole depth in hole table.
PR 1538024: Fixed a problem where importing DXF files into Solid Edge are scaled by a factor 1/1000
PR 5395786: Fixed a problem where exporting a draft file to DXF is extremely slow when saving to a mapped drive

=====Insight=====

PR 1523670: Fixed a problem where the LCA Path does not point to a subsite
PR 5563703: Fixed a problem with an incorrect error saying Foldermap is read only during ATL
PR 1534454: Fixed a problem where permissions have been modified on doc lib
PR 1534720: Fixed a problem where links cannot be resolved after LCA has been used multiple times
PR 5534235: Fixed a problem where the V18 viewer is not displaying files via an erp system
