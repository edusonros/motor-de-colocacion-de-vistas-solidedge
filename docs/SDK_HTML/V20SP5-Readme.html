PR 5882948: Fixed a problem where an imported NX assembly is missing parts

PR 2091832: Fixed a problem where opening a JT assembly aborts
PR 1617474: Fixed a problem where Save As causes an Include - link to be deleted in the new assy

PR 5214177: Fixed a problem where making a subasm adjustable places patterns in wrong location

PR 5849418: Fixed a problem where editing protrusion size twice in a row results in an incorrect solve.

PR 1641535: Fixed a problem where Ideas TDM Migration is very slow
