Version:        20.00.08.04

    • Windows XP Professional® operating system (32 bit or 64 bit)
    • Windows Vista Business® or Vista Enterprise® operating system (32 bit or 64 bit)
    • Internet Explorer 7.0 (IE 6.0 minimum)

This Update addresses the following Customer Reported PRs:
----------------------------------------------------------

=====Assembly=====

PR 5847501: Fixed problem that caused an abort when applying an angular relationship
PR 5906839: Fixed a problem where the Assembly Pattern command is creating extra occurrences
PR 1665948: Fixed a problem where parts were not listed in Edgebar and cannot be deleted
PR 2098485: Fixed a problem that caused an abort when opening Physical Property Manager
PR 1625279: Fixed a problem where Interpart-Manager lists the wrong information
PR 2097711: Fixed a problem where the mass of physical properties is not correct
PR 1664488: Fixed a problem where you cannot generate a standard part
PR 5969985: Fixed a problem that caused an abort based on auto-submitted crashlogs
PR 2097514: Fixed a problem where FlashFit surfaces are locating in incorrect order
PR 5971808: Fixed a problem that caused unpredictable arc dimensions in XpresRoute
PR 5972224: Fixed a problem where helical spur gears are being created incorrectly
PR 1623215: Fixed a problem where the "Edit Adjustable Part" command was missing in Solid Edge Foundation.
PR 5728742: Fixed a problem where you cannot create a simplified assembly
PR 5984610: Fixed a problem where Frames cannot be seen in a simplified assembly saved as a part.
PR 2098304: Fixed a problem where editing a tube radii returns to default after Transfer
PR 5964249: Fixed a problem with performance and memory issues editing and updating a weldment.
PR 5784262: Fixed a problem with When placing simplified assy, components disappear after dragging into graphics
PR 1662218: Fixed a problem with a Standard parts error, unsupported format of a non-CAD document
PR 1668572: Fixed a problem where you are unable to change frame end conditions to butt2
PR 2090384: Fixed a problem where Save As RTF format garbles Japanese text
PR 5988608: Fixed problems with assembly feature performance.

=====Part=====

PR 2102153: Fixed a problem that caused an abort when setting the maximum density is 0.5
PR 5825612: Fixed a problem where UG assembly file does not translate correctly in Solid Edge V20.
PR 1662715: Fixed a problem that caused an abort after update all.
PR 2101442: Fixed a problem where you cannot create Contour Flange when setting Bend radius=0mm
PR 1672651: Fixed a problem where you are unable to flatten a sheet metal part with a chamfer
PR 1661859: Fixed a problem where Model view takes a very long time in View and Markup.
PR 5785898: Fixed a problem where Solid Edge aborts opening a Parasolid file.
PR 1668264: Fixed a problem with open JT file causing duplicate the file names.

=====Draft=====

PR 1639511: Fixed a problem where dimensions are not correct when opening a DXF file.
PR 2098559: Fixed a problem where dimensions are missing in imported dxf
PR 5957864: Fixed a problem that caused an abort during create/update of a view with a PMI model view.
PR 5786340: Fixed a problem where the Elements on the Auto Hide layer are being exported as visible
PR 2101520: Fixed a problem where views with old tubes failed to place correctly.
PR 2103451: Fixed a problem where a "Surface Texture Symbol" cannot be exported to DXF.
PR 2101453: Fixed a problem where dimension and lines overlap when exported to DXF
PR 5802490: Fixed a problem where AutoCAD Ft-In dimensions do not translate properly
PR 5991964: Fixed a problem with "Error: no reference" displayed in callouts linked to missing properties

=====Insight=====
 
PR 1662107: Fixed a problem where create in place does not warn user in certain scenarios
PR 5941878: Fixed a problem where a new part is left in a checked out state when draft is checked in and closed
PR 1637467: Fixed a problem where Version comments (Edit Profile) are not listed in version history in SharePoint
PR 5957759: Fixed a problem where older files from cache overwrite newer files on Insight/SharePoint server

=====API=====
 
PR 5979480: Fixed a problem where DraftDataAPI.dll had a dependency on SEAlloc.dll
PR 1615362: Fixed a problem with API-open/close-events causing corrupt pointers
PR 5959193: Fixed a problem with the compatibility of SEPartReader.DLL on Vista
