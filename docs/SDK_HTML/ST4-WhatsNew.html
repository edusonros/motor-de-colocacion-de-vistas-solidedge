What's New > Solid Edge ST4 - What's New

Solid Edge ST3 introduced significant changes in the Synchronous environments. The SolidEdgeAssemblySync and SolidEdgePartSync type libraries introduced in Solid Edge ST have been obsolete and are completely removed in Solid Edge ST4.  Please review the Solid Edge ST3 for important details.
