Specifies the number of Curve objects used in the construction of the swept feature.
Contains Edge and/or Profile objects. The size of the array is equal to the value of NumCurves.
Contains members of the FeaturePropertyConstant set (igProfileBasedCrossSection and igEdgeBasedCrossSection) that specify whether each corresponding member of the TraceCurves array is an Edge or Profile object. The size of the array is equal to the value of NumCurves.
Contains the points that define the start points of the corresponding cross sections. Each point is represented as an array of two Doubles (for example, Dim dblOrigin(2) as Double; Dim Origins() as Variant). For members of this array that correspond to periodic cross sections (circles and ellipses), specify a value of 0 instead of an array that represents a point.
