Release Notes:
--------------

Product:        Solid Edge
Version:        16.00.03.04
Date:           08-Dec-2004


This release of Solid Edge has been certified to run on the following:
 • Windows 2000 Professional® operating system running Service Pack 4.
 • Windows XP Professional® operating system running Service Pack 1 or 2.
 • Internet Explorer 6.0 or higher


This Update should be installed on top of Solid Edge Version:
16.00.00.63, 16.00.01.05, 16.00.02.04


--------------------
End of Release Notes
--------------------

------------------------------------
This Update addresses the following:
------------------------------------

=====Assembly=====

PR 1352922: Fixed a performance problem with open/Close&Return.
PR 2042925: Fixed a problem with honoring the as-last-saved setting when applying a config.
PR 4911514: Fixed a problem with minimum distance, 'Solids Only' is returning incorrect values.
PR 5024226: Fixed a problem losing part modifications when opening a parent assembly.
PR 5033170: Fixed a problem with excluded occurrences sometimes not appearing for Alternate Assemblies.
PR 5044465: Fixed a problem with FOA styles.
PR 5059535: Added a warning when using delivered files within the Frame command on a local machine.
PR 5070608: Fixed a problem with new parts being shown by existing configs.
PR 1364040: Fixed a problem that caused a crash when opening a specific assembly.
PR 1357488: Fixed a problem with SimplyMotion under Design&Drafting license.
PR 2043284: Fixed a problem where executing save as into jt format more than two times fails.
PR 5069195: Fixed a problem to address an assembly solve failure.
PR 4980664: Fixed a problem with Show All while in-place active from Draft.
PR 5083073: Fixed a problem using mirror assembly with sheet metal parts.
PR 5096408: Fixed a problem placing JIS standard parts on a Japanese system.
PR 1369165: Fixed a problem where assembly driven part features fail with patterns of holes
 
=====Part=====

PR 5075766: Fixed a problem that cuased a rash when exiting draft
PR 5076054: Fixed a problem that caused a crash while editing a delete simplified feature.
PR 1359559: Fixed a problem with the display of an included sketch
PR 5073928: Fixed a problem where, Property is incorrectly filled in Prop Mgr
PR 5077136: Fixed a problem where density is no longer being overwritten from other environments.
PR 1359591: Fixed a problem with dangling dimensions of holes
PR 1365676: Fixed a problem when editing profile text.
PR 5047639: Fixed a problem where cannot set the thickness of a sheetmetal tab if a construction part copy exists
PR 5077058: Fixed a problem where features failed after editing an inserted weldment feature
PR 1366400: Fixed a problem to prevent a crash based on Crashlog data.
PR 1366809: Fixed a problem to prevent a crash when displaying axis of revolution.
PR 1366853: Fixed a problem with the displayed value of Density
PR 1367879: Fixed a problem with the displayed value of Density.
PR 2043676: Fixed a problem to correct ColorManager anomalies.

=====Draft=====

PR 1325511: Fixed a problem with a condition that sometimes caused drawing views to disappear on update.
PR 1363841: Fixed a problem where Create 3D doesn't work with D&D license.
PR 2042776: Fixed a problem with surface texture symbols disappearing
PR 5031707: Fixed a problem where the dimension is not getting deleted when the parent element is deleted
PR 5042779: Fixed a problem where grid is not always accurate when using "Snap to" option.
PR 5044930: Fixed a problem where re-attached coordinate dimensions stay sick color.
PR 5068709: Fixed a problem where dimensions are changing when different family members are applied.
PR 5074172: Fixed a problem where parts list balloons are changed when opening a file without the model present.
PR 5076518: Fixed a problem where edge painter did not work correctly with bend centerlines.
PR 1363936: Fixed a problem where geometry from the 2dModelSheet couldn't be copied and pasted
PR 2035244: Fixed a problem where some of the group dimensions increasing in height when copy-pasted
PR 5076051: Fixed a problem where custom center line style is using the wring line type since V16
PR 5081511: Fixed a problem where SAT data is not importing
PR 5083019: Fixed a problem importing a dwg file created by solid edge not trimming curves correctly
PR 1364577: Fixed a problem of importing DXF files created by ME10
PR 5046787: Fixed a problem where rectangular drawing view borders were incorrectly printing
PR 5081143: Fixed a problem that caused a crash when editing a particular balloon
PR 5025565: Fixed a problem with copy draft view geometry into 2D model view.
PR 5086154: Users will not be allowed to save Draft files while editing a cutting plane in that file.
PR 5096162: Fixed a problem where thread graphics were not update correctly when no edges were shown.
 

=====API=====

PR 1360611: The SetNode, SetPole and SetKeyPoint methods on the bspline curve 2d object will now mirror the UI keypoint drag behavior.
PR 1351060: Fixed a problem where the SolidEdgeDraft.ModelMember.ComponentType automation API returned an error code when the component is a tube centerline.

=====Insight=====

PR 5073914: Fixed a problem when performing actions in where-used results pane of Revision Manager
PR 1366297: Fixed a problem that caused an abort when switching to V&M in the middle of entering a new filename
PR 4946563: Fixed a problem with SharePoint workflow not working on regional language servers
PR 5072826: Insight for Design & Drafting license is now fully-enabled.
PR 5084779: Fixed a problem where an old docid was being reused.
PR 5085807: Fixed a problem of PCF getting corrupt after moving and editing by different users
PR 5053109: Fixed a problem where a reader opens a file which is already checked out by someone else then we were currently showing the error message that user does not have the privileges. Changed the implementation to show an error message that the file has been checked out to a different user.
PR 5062198: Fixed a problem while performing the replace operation, we getting a warning message "Could not delete the file" for a file which does not exists in the cache.
