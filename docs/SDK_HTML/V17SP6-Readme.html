Release Notes:
--------------

Product:        Solid Edge
Version:        17.00.06.03
Date:           27-Oct-2005


This release of Solid Edge has been certified to run on the following:
 • Windows 2000 Professional® operating system running Service Pack 4.
 • Windows XP Professional® operating system running Service Pack 1 or 2.
 • Internet Explorer 6.0 or higher


This Update should be installed on top of Solid Edge Version:
17.00.00.65, 17.00.01.06, 17.00.02.04, 17.00.03.04, 17.00.04.03, 17.00.05.03


--------------------
End of Release Notes
--------------------

------------------------------------
This Update addresses the following:
------------------------------------

=====Assembly=====

PR 1420619: Fixed a problem where you are unable to pattern after placing an Assembly Feature
PR 1449109: Fixed a problem where an imported ACIS file was generating unusable filenames
PR 2052865: Fixed a problem where the position of an adjustable subassembly changes when opened.
PR 2053523: Fixed a problem where you can not select construction Display button
PR 5306669: Fixed a problem where Parts stay transparent after applying relationships, then can't be hidden.
PR 5325821: Fixed a problem where Parts with # in names lose # and previous characters in Reports.
PR 1449789: Fixed a problem that caused a crash during 'save as image' if coordinate systems are visible
PR 2054556: Fixed a problem that caused an abort when clicking assembly relationship in assembly pathfinder
PR 5332875: Fixed a problem that caused a crash when printing an Assembly file.
PR 2054706: Fixed a problem that caused a crash in JDocument::IOleInPlaceViewsImp::SetActive3DObject.
PR 5332702: Fixed a problem where certain part files would no longer open after an update all and saving an assembly file.
 
=====Part=====

PR 2052480: Fixed a problem where Fujitsu Celsius N430 & J340 crash in command finder.
PR 5223232: Fixed a problem where memory usage goes up as view is rotated if coordinate systems are on.
PR 5321263: Fixed a problem when Display lists have been enable for ATI FireGL graphics cards.
PR 5326203: Fixed a problem with the old ugsplm.com crashlog email address.
PR 5327404: Fixed a problem where you are unable to stitch surfaces imported from NX
PR 2054441: Fixed a problem that could cause an abort based on crashlog data.
PR 5331840: Fixed a problem where a simple part would not flatten after adding flanges.

=====Draft=====

PR 1448552: Fixed a problem with shaded drawing views.
PR 1448976: Fixed a problem where parts in drawing views would show in the wrong position.
PR 5225327: Fixed a problem with embedded Word documents not updating after edit.
PR 5327883: Fixed a problem where File/Open Preview doesn't work with pre-V14 files
PR 5331285: Fixed a problem where attribute text is displaying after translation of DWG file
PR 1449611: Fixed a problem that caused a crash during section view creation

=====API=====

PR 2054505: Fixed a problem where APIs accessing part storage fails.
PR 1446277: Fixed a problem that corrects behavior of AddDraftView automation API.

=====Insight=====

PR 1447004: Improved performance during a window change from ASM to DFT within an Insight environment.
