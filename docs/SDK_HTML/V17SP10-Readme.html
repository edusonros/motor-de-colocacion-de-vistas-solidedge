Release Notes:
--------------

Product:        Solid Edge
Version:        17.00.10.02
Date:           03-Mar-2006


This release of Solid Edge has been certified to run on the following:
 • Windows 2000 Professional® operating system running Service Pack 4.
 • Windows XP Professional® operating system running Service Pack 1 or 2.
 • Internet Explorer 6.0 or higher

This Update can be instaled installed on top of Version:
17.00.00.65, 17.00.01.06, 17.00.02.04, 17.00.03.04, 17.00.04.03, 17.00.05.03,
17.00.06.03, 17.00.07.03, 17.00.08.03, or 17.00.09.03


Note: Updates are all inclusive and can be installed on top of the base release
or any released UPdate.

End of Release Notes

------------------------------------
This Update addresses the following:
------------------------------------

=====Assembly=====

PR 1476826: Fixed a problem where relationships are not updating after editing parts.
PR 2062434: Fixed a problem where you can't restore the auto backup file in V17
PR 5384474: Fixed a problem where Printing Reports truncate column titles on Jpn version
PR 1502640: Fixed an abort in FlashFit when hitting Flip after a circular dependency error.
PR 5371628: Fixed an abort when deleting a relationship from ribbon bar while in Assembly using Reduced Input mode.
PR 5407218: Fixed a problem where in the assembly file fails to open.
PR 1499431: Fixed a problem with locating target faces on subassembly parts for traditional relationships.
PR 1501482: Fixed a problem with not being able to select faces when using FlashFit in Edit Definition.
PR 1503322: Fixed a problem where Part modifications are reverting when a display configuration is applied.
PR 2060309: Fixed a problem where you are unable to select parts from view
PR 2062548: Fixed a problem where some assembly files would fail to open.
PR 2063432: Fixed a problem with restoring a backup file to a network location.
PR 5347858: Fixed a problem where Assembly features were not reflected at the toplevel
PR 5353329: Fixed a problem where non-auto dimensions were getting hidden when they should not be.
PR 5373421: Fixed a problem where adding a horizontal alignment in hole 28 causes holes to fail.
PR 2060417: Fixed a problem where a Model does not show up
 
=====Part=====

PR 1466602: Fixed a problem where a Sheet metal part is not being computed when the material name is changed
PR 1494255: Fixed a problem where SE crashed during flatten sheet metal part
PR 2062575: Fixed a problem where Material table APIs crashed
PR 1471984: Fixed a problem where flat pattern fails when recomputing a V14 part ( regression)
PR 5210070: Fixed a problem where a Flyout toolbar button does not show custom bitmap.
PR 5391641: Fixed a problem when loading Parasolid files using Solid Edge API

=====Draft=====

PR 1493661: Fixed a problem where non-rectangular cropping in V15 becomes uncropped when opened in V16 or later.
PR 5393644: Fixed a problem where the Author Field is getting set back to login name when Auto-save is on.
PR 1494809: Fixed a problem where sketches from an assembly shown in a drawing view disappear on update.
PR 2063758: Fixed a problem that caused some V16 files unable to open in V17 or V18.

=====API=====

PR 2063313: Fixed a problem where custom properties are not accessible using propauto for FOA members

=====Insight=====

PR 5388175: Fixed a problem where Add to Library process is not completing when defining a folder contents
PR 5397191: Fixed a problem where you are unable to Save or use Add to Library for a particular part file that is 135 Mb.
PR 5401669: Fixed a problem where Solid Edge search is crashing the w3wp.exe (SharePoint) process.
