Solid Edge Draft Type Library : CuttingPlane Object
Public Methods
 CreateViewThis method will create an empty (un updated) view. There will be one optional Boolean argument to specify that the view should be a revolved section, and it will default to false. Note that the fold segment will default to the first segment, but can be changed with the FoldSegment property.
 GetCaptionPositionReturns the x and y co-ordinates of the cutting plane's caption in sheet space.
 GetFoldLineWithViewDirectionReturns the start position, end position and the view direction of the section line. In case of multiple segments in the section line, the positions correspond to the end points of the active segment.
 SetCaptionPositionSets the x and y co-ordinates of the cutting plane's caption in sheet space.

 ThickLineLengthRatioSets and returns multiplication factor for the line length. Ex. line length = ThickLineLengthRatio x font size.
 ViewSheetNumberDefinitionTextSets and returns the View Sheet Number (%VN) definition string. This is intended for use with %{View Sheet Number|DV} property text and may include other formatting options.
 ViewSheetNumberLocationSets and returns the Sheet Number Location of the cutting plane.
 ViewSideSets or returns the side of the free end of the fold segment that determines the direction of the view. This will use the existing enumerated type SectionViewProfileSide, with only igLeftProfileSide and igRightProfileSide being valid. If the view has not been created it will have a default value of igLeftProfileSide.
