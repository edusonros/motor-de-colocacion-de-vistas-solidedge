Solid Edge Assembly Type Library : AssemblyDocument Object
Description
A document type used exclusively for assemblies.
Object Model
Remarks
The AssemblyDocument object is a container for OLE links to other documents that contain occurrences (parts or other assemblies). An assembly document is a specific type of document that is used exclusively for assemblies, and it has its own OLE automation interface. This object allows you to place occurrences into an assembly and to examine existing occurrences and their relationships.
