----------------------------------------------------------
This Update addresses the following Customer Reported PRs:
----------------------------------------------------------

=====Assembly=====

PR 1633466: Fixed a problem where the Adjustable Part dialog appears only once
PR 1642090: Fixed a problem where tubes are not updating properly inside a FOA file.
PR 5918490: Fixed a problem where Assembly features are taking a long time to create/update in this file.
PR 2090861: Fixed a problem where Solid Edge aborts when double clicking on an X_T file on desktop
PR 1648907: Fixed a problem where the Preview is not updating in Property Manager

=====Part=====

PR 5875755: Fixed a problem where a large Step File is not translating properly
PR 5864044/2095252: Fixed a problem where files are not saved even though "save" is selected on the memory warning dialog
PR 5864044/1628784: Fixed a problem where some features are not updating after Editing Profile
PR 1626397: Fixed a problem where Solid Edge crashes when editing a pattern.
PR 5923272: Fixed a noticeable performance difference between V18 and V20 when creating an extrusion in-place active
PR 5796903: Fixed a problem where Solid Edge would crash when deleting a line from profile in a Tab.
PR 5876754: Fixed a problem where Solid Edge was taking a long time to validate/checkout a second and third license
PR 2084055: Fixed a problem where resizing a hole in the is file caused Solid Edge to lock up.
PR 1637173: Fixed a problem where an Unbend feature modifies existing tabs located in a bend zone
PR 1619304: Fixed a problem where Solid Edge closes after editing a text profile within a cutout.
PR 5852431:  Fixed a problem where tab creation results in multiple faces.

=====Draft=====

PR 5901846: Fixed a problem where Solid Edge crashed when creating Detail View
PR 5911708: Fixed a problem where a Hole callout was wrong in section view.
PR 1651030: Fixed a problem where a User table was corrupted after rows were deleted.
PR 1638310: Fixed a problem where DWG and DXF files were not generated correctly.
PR 1635223: Fixed a problem where a drawing view of a model with simplified subassemblies did not update correctly
PR 1640473: Fixed a problem where copying a drawing view w/assembly overrides caused subsequent views to be placed incorrectly.
PR 5930003: Fixed a problem where Solid Edge was crashing when trying to create a section view from a detail view.
PR 5932877: Fixed a problem where a drawing view of an Assembly created erroneous geometry
PR 1646959: Fixed a problem where revolved section views became corrupt after an independent detail view was deleted.
PR 1645262: Fixed a problem where copy/paste between documents was very slow
PR 2097038: Fixed a problem where dimension values in this file displayed incorrectly

=====Insight=====
 
PR 5831605: Fixed a problem where a PCF file would not display in View and Markup
PR 1649542: Fixed a problem where the BOM WebPart doesn't properly export text which contains a comma.
