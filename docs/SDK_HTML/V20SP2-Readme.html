----------------------------------------------------------
This Update addresses the following Customer Reported PRs:
----------------------------------------------------------

=====Assembly=====

PR 5827283: Fixed a problem where Explode Config used for ERA goes bad after adding a motor and entering animation
PR 2088108: Fixed a problem where Physical Props are not reporting unassigned densities correctly selected parts.
PR 1580928: Fixed a problem with the positioning of hole assembly feature.
PR 5588898: Fixed a problem where a section view is causing file size increase after a save.
PR 2088423: Fixed a problem where the assembly ribbon flickering when user is Closing & Returning to an expanded asm.
PR 1609662: Fixed a problem where the user is unable to delete pipe occurrences.
PR 5815906: Fixed a problem where Solid Edge is crashing when you try to save out an empty simplified assembly.
PR 2083366: Fixed a problem where Section cut faces are not updating properly.
PR 5828903: Fixed a problem where you can not select a silhouette edge of one part.
PR 5765837: Fixed a problem where Export to JT did not honor Solid Edge Occurrence Properties
PR 1579648: Fixed a problem where entries from custom.xml are not used and cannot be deleted
PR 5831498: Fixed a problem with assembly file-open performance

=====Part=====

PR 2089001: Fixed a problem where editing a cutout profile and finishing without changes results in a recompute
PR 5822624: Fixed a problem where you are unable to get a contour flange around an internal octagon
PR 5747437: Fixed a problem where a symmetry relationship not be created
PR 1624643: Fixed a problem where German Locale shows read/write fields incorrectly in variable table

=====Draft=====

PR 1617165: Fixed a problem where sheets are missing when using Save as DXF
PR 5822573: Fixed a problem where you are unable to Copy/Move multiple elements in Draw in View.
PR 5827506: Fixed a problem where this Draft file does not open.
PR 5831180: Fixed a problem where you are unable to create a PDF dataset.
PR 5835355: Fixed a problem where this Draft file does not open.
PR 1619145: Fixed a problem where bend table column headers are not displayed.
PR 5622219: Fixed a problem where dangling dimension have the wrong value when reattached

=====Insight=====

PR 5832513: Fixed a problem where there was no Obsolete option in the function "Move documents through workflow"
PR 5831605: Fixed a problem where you are unable to display a pcf file in the View and Markup environment.
