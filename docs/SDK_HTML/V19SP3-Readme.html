Date:           18-Jan-2007

This release of Solid Edge has been certified to run on the following:
    • Windows XP Professional® operating system (32 bit or 64 bit)
    • Internet Explorer 6.0 or higher

----------------------------------------------------------
This Update addresses the following Customer Reported PRs:
----------------------------------------------------------

=====Assembly=====

PR 1550966: Fixed a problem where Standard Parts does not create selected part
PR 5532555: Fixed a problem where Sub-Assembly display configurations cause all parts to be hidden.
PR 5608103: Fixed a problem where automatic frame component positioning does not work
PR 5631004: Fixed a problem where section view FF show lines and a circle that should not be there.
PR 2073663: Fixed a problem where the Position3d constraint fails to move part after modifying sketch that drives it
PR 5656020: Fixed a problem where Section Views in assembly are not cutting parts properly for symmetric through-all
PR 5657836: Fixed a problem where you can't place variable length standard parts after installing update 2.
PR 5648138: Fixed a problem with the default output folder path when opening a UG file
PR 5656912: Fixed a problem where Solid Edge crashes when creating an assembly pattern
PR 1531552: Fixed a problem where a CAM relationship creates interference between parts.
PR 1557045: Fixed a problem with performance in assembly
PR 1564217: Fixed a problem where clicking on a triad origin dismisses triad while placing dimension.
PR 5624749: Fixed a problem where the STL translator is not trimming surfaces correctly.
PR 1545313: Fixed a problem where editing a section view is not possible in Design & Drafting and Solid Edge Foundation
PR 5660221: Fixed a problem where an IGES file translation results in missing parts.

=====Part=====

PR 5624183: Fixed a problem where changing the BlueSurf option to Tangent causes Solid Edge to crash
PR 1560496: Fixed a problem with STEP files that were created with a comma as the decimal delimiter
PR 5582087: Fixed a problem where you are "Unable to create body" using mirror sketch curves to create Dimple
PR 5610135: Fixed a problem where Undo results are incorrect after making profile changes in Part.
PR 5660666: Fixed a problem where print from wireframe mode in Profile does not include profile geometry.
PR 5660770: Fixed a problem where the 2nd horizontal chained dimension gets placed incorrectly
PR 5669631: Fixed a problem where Solid Edge crashed after recomputing a Flat Patten
PR 2066862: Added a tool that allows links to non-existent FOP and divide part member files to be removed.
PR 5677117: Fixed a problem where Move Face with keyin and Return terminates the command.
PR 5683208: Fixed a problem with When a attempt to convert using SEPROE.exe a license error occurs.

=====Draft=====

PR 5633556: Fixed a problem where a Parts list would not stay up to date.
PR 5618414: Fixed a problem where old DQ views were updating incorrectly.
PR 1554625: Fixed a problem where a Draft file opens in V16 and V17, but not in V18 or V19
PR 1557190: Fixed a problem where complex strings are not displaying in block occurrences.
PR 5659161: Fixed a problem where Update 2 scales dimensions to the views scale on export to DXF
PR 5593722: Fixed a problem where copy and paste title block results in wrong size text/lines
PR 1560113: Fixed a problem where you are unable to open this DXF file in V19
PR 2075329: Fixed a problem where the display of a weld reinforcement symbol is not correct
PR 5650136: Fixed a problem where a DXF file created by Solid Edge cannot be opened in AutoCAD
PR 5666550: Fixed a problem where Copy and Paste changes text fonts
PR 5671002: Fixed a problem where the GDT font aspect ratio is not set to 1 when opening an AutoCAD file
PR 5657881: Fixed a problem where Solid Edge crashes when placing a distance between dimension

=====API=====

PR 1553359: Fixed a problem with Frames API
PR 1553510: Fixed a problem with GetOrientation for StructuralFrame
PR 5389793: Fixed a problem where an application's locale changing when loading separtreader.dll.

=====Insight=====

PR 1551714: Fixed a problem where performance on file open very slow in V18 Update 8 compared to V16 Update 9.
PR 1557352: Fixed a problem where ServerMap.txt doesn't work - Insight still tries to access the old server
PR 1560300: Fixed a problem where LCA Dialog doesn't update the file list for the SharePoint doclib folder
PR 5613700: Fixed a problem where Solid Edge search will not search document libraries in sub sites
PR 5648569: Fixed a problem where you cannot open and revise on the fly from internet explorer dashboard
PR 5651802: Fixed a problem where Create in Place command fails when using a FQDN naming convention for the URL.
PR 1554819: Fixed a problem where validating searchscope entries takes approx. 20 seconds.
PR 5660196: Fixed a problem with performance when searching large number of doclibs
PR 1562958: Fixed a problem where search doesn't return a result when the user doesn't have access to a document library
PR 1563289: Fixed a problem where a search never ends when "Look in" the top level site is specified
PR 5613700: Fixed a problem where Solid Edge search will not search document libraries in sub sites
PR 5659034: Fixed a problem where the File Open dialog does not remember the last used folder.
PR 5670057: Fixed a problem where 50 or so files were deleted from server and local cache
