    • Windows XP Professional® operating system (32-bit) running Service Pack 2

------------------------------------
This Update addresses the following:
------------------------------------

=====Assembly=====

PR 5365270: Fixed a problem that will avoid potential crash situations during a display event.
PR 2060266: Fixed a problem where some parts were activating when an assembly was opened with the all-parts-inactive.
PR 5345754: Fixed a problem where SE disappears creating fastener system in managed assembly after IPA into part.
PR 5367968: Fixed a problem to correct a performance issue with graphics card driven displays.
PR 5368953: Fixed a problem that caused a crash on 'Revision' menu of unmanaged part inserted in managed file.
PR 5368558: Fixed a problem where you cannot save as plmxml when asm weldments and protrusions are present

=====Part=====

PR 2060125: Fixed a problem where undoing during the edit of a loft caused the document to close.
PR 5366336: Fixed a problem where the fillet command can cause an abort when drawing across the fillet corner.
PR 1447589: Fixed a problem that caused an abort when trying to make changes within property manager
PR 1476638: Fixed a problem where an abort can occur due to changes in a bspline curve
PR 2060303: Fixed a problem where driving dimension cannot be created in existing profile.
PR 2060313: Fixed a problem where adding draft is not highlighting in face step with failed parting geometry.
PR 5327202: Added support for reading NX4 files.
PR 5359436: Fixed a problem with imported styles not being exported correctly to Parasolid
PR 5359160: Fixed a problem where constraints were flipping when a series of solves contains a failed solve.

=====Draft=====

PR 5366751: Fixed a problem that caused Draft files to fail to Save after updating drawing views.
PR 1479509: Fixed a problem where that caused a crash using the Select Tool
PR 1478451: Fixed a problem where groups with projection lines are behaving strangely
PR 5364014: Fixed a problem where a Weld bead with fractional round off gets incorrect weld symbol in Draft.
PR 5365221: Fixed a memory overwrite problem in printing that may have caused crashes
PR 5368249: Fixed a problem where SE could not detect if Czech STEP files are assembly or part
PR 1401017: Fixed a problem where Symbols with groups lose group names when inserted as geometry
PR 4673160: Fixed a problem where you could not stretch a named group of elements with the stretch command
PR 5350928: Fixed a problem where a Label weld cross section is reset to zero.
PR 5366571: Fixed a problem where hatch display was incorrect.
PR 5370044: Fixed a problem that prevented turning off the display of bend centerlines.
PR 5371932: Fixed a problem with counterbore size and counterbore depth.
PR 5372124: Fixed a problem that would cause existing Draft files to fail to open in V18.
PR 5371700: Fixed a problem where spreadsheet data copied into the Windows clipboard from Excel and pasted into Solid Edge Draft was resulting in the creation of an image object with a bitmap of the spreadsheet cells, instead of an editable spreadsheet cell within the .DFT file.

=====API=====

PR 5365704: Fixed a problem that caused a crash in CUnknownEnumDataHolder::SetData.
PR 5368064: Fixed a problem where .NET addins fail to add their menus and toolbars.
PR 2059928: Enhancements have been made to the Application Display Automation controls to allow programmatic selection of Application Display settings.
PR 5362288: Properties and methods have been added to the assembly automation objects SegmentDistanceRel ation3d, SegmentRadiusRelation3d, SegmentAngularRelation3d and SegmentTangentRelation3d so that additional information about these objects can be obtained. The method GetPathSegmentsRelations has been added to the Tube automation object so that the segment relations of a tubes path can easily be found.

=====Insight=====

PR 1481372: Fixed a problem where you can now revise Sheet Metal FOP file. (Need to Add to Library again.)
PR 5370924: Fixed a problem where Add To Library aborts when importing a specific file to Insight.
PR 5330720: Fixed a problem where Cannot save a draft file because SE complains of a missing assy file
PR 5371185: Fixed a problem where "Add To Library failure due to a file whose links cannot be read".
PR 5371200: Fixed a problem where Insight operations failed after using Delete All command.
PR 1468781: "#" will now be allow to be set as revision delimiter for non-Insight files only.
PR 5347635: Fixed a problem where Property Manager does not list any FOP Children when in the master file and properties can not be modified for that member in case of managed files.
 
=====Embedded Client=====

PR 5365862: Fixed a problem that caused a crash when opening an Assembly with Weldment Features.
PR 5366672: Fixed a problem that caused a crash when pushing BOM button on open dialog after Bad Connection Warning.
PR 5354838: Fixed a problem where TcE Released status was shown for assembly files and not the parts
PR 5357793: Fixed a problem where Sort order incorrect after doing a delete from cache.
PR 5358513: Fixed a problem with Revision Rules, will now be Listed only Once in File Open -> BOM View dialog.
PR 5361225: Fixed a problem where you are Unable to open earlier revisions.
PR 5361378: Fixed a problem where Formula was not displayed when Status in pathfinder is turned on
PR 5361663: Fixed a problem where a Warning shown "Cannot be deleted" when Undo Check-out applied
PR 5365842: Fixed a problem where the connection List is sometime not expanded when dialog is Resized.
PR 5366651: Fixed a problem where incorrect Item Type was displayed for an assembly file during specified workflow
PR 5367423: Fixed a problem where EM_transient_connection_only = ON allows creation and removal of connections
PR 5367444: Fixed a problem where Undo Checkout on Multiple Items selected in Cache Assist. don't Work.
PR 5367458: Fixed a problem where the Fie->Revisions option will be disabled for the newly created document.
PR 5369609: Added a new option "subfolders" that will be available in the Teamcenter Bulk Loader.
PR 5364574: Fixed a problem that caused a crash in Solid Edge Embedded Client.
PR 5374076: Fixed a problem with Transient Connection: Opening document from Cache Assistant Switch to nonTC Mode
PR 5369612: Fixed a problem where revising the part having a draft file residing under same revision does not crash.
PR 5349044: Fixed a problem where Upload action was causing the document to checkin
PR 5366630: Fixed a problem where pushing Restore on Property Dialog was setting Item Type incorrectly
