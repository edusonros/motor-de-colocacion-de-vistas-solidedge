Version:        19.00.05.03
Date:           19-Apr-2007

This release of Solid Edge has been certified to run on the following:
    • Windows XP Professional® operating system (32 bit or 64 bit)
    • Internet Explorer 6.0 or higher

This Update can be instaled installed on top of Version: 19.00.00.66, 19.00.01.06,

----------------------------------------------------------
This Update addresses the following Customer Reported PRs:
----------------------------------------------------------

=====Assembly=====

PR 1581284: Fixed a problem where a message was not displayed when a SaveAs failed on a write-protected part.
PR 1582000: Fixed a problem where simplified assembly display is clipped
PR 1583107: Fixed a problem where changes made to a sketch were lost after saving assembly
PR 2076006: Fixed a problem with the creation of physical conductors.
PR 5619983: Fixed a problem where assembly parts were getting ?'s in pathfinder indicating missing components.
PR 5687761: Fixed a problem where Property Text gives bad data for simplified assemblies in PMI.
PR 5703102: Fixed a problem where changes made in Edit Table are not retained
PR 5663102: Fixed a problem where a strange circle displays in this drawing view
PR 5715979: Fixed a problem where Solid Edge displays a "Solid Edge requires additional memory to... " error
PR 1533732: Fixed a problem where Assembly pattern positions parts incorrectly
PR 5665884: Fixed a problem where Assembly hangs replacing a part
PR 5683521: Fixed a problem where you cannot replace an alternate sub-assembly.
PR 5600044: Fixed a problem where parts are going over constrained after creating a new FOA member using the table editor

=====Part=====

PR 1580247: Fixed a problem where Solid Edge crashes when normal cutout is used for a logo
PR 2078811: Fixed a problem where "Bend relief" is not correctly displayed.
PR 5709603: Fixed a problem where the Family Identifier message when editing threaded holes is misleading
PR 5218427: Fixed a problem where you cannot edit text profile when anchored to an arc

=====Draft=====

PR 1582140: Fixed a problem where Solid Edge aborts when clicking on the Options button when exporting to DWG
PR 1545609: Fixed a problem where image from a draft file are removed when saved.
PR 1583952: Fixed a problem with thread display at chamfer
PR 5524235: Fixed a problem where clip art pasted from word is displaying incorrectly.
PR 5681877: Fixed a problem where 3D DWG files do not import as SAT from ACAD 2004
PR 5722213: Fixed a problem where incorrect dimension values show in acad when a draft is exported
PR 1583575: Fixed a problem where saving a DWG or DXF in TEMP folder produces an error
PR 1586550: Fixed a problem where brackets of a reference dimension will be printed in a lighter color.
PR 1586577: Fixed a problem where Solid Edge V19 Update 4 imports the AutoCAD Color 7 as white
PR 2080071: Fixed a problem where imported line styles from DWG incorrect.
PR 2080212: Fixed a problem where linear dimensions export with no picture data

=====API=====

PR 1573535: Fixed a problem with threads in API in a C# program
 
=====Insight=====

PR 1563966: Fixed a problem where Search functionality is not working correctly when the user searches on a Property
PR 1579038: Fixed a problem where the checked out cfg files cannot be checked in
PR 2079055: Fixed a problem where 'Part Name' Japanese strings are garbled in Pathfinder
PR 5710686: Fixed a problem where Long File names are not displayed completely on Solid Edge close dialog
PR 5357907: Fixed a problem where the Revision command was not handling an FOA revision correctly
PR 5726666: Fixed a problem where FOP Master file is crashing in Rev Manager when renaming
