Solid Edge Draft Type Library : SheetWindow Object
Description
A 2-D view type that displays sheets and has a tab user interface.
Object Model
Remarks
When a SheetWindow object is accessed, it is usually as a superset object called a View. If the window type is SheetWindow, then the methods and properties of the SheetWindow object extend to any standard Window object methods or properties.
