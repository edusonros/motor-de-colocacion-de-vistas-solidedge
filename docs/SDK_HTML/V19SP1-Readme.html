----------------------------------------------------------
This Update addresses the following Customer Reported PRs:
----------------------------------------------------------

=====Assembly=====

PR 5582013: Fixed a problem where Solid Edge V19 fails to run.
PR 1546531: Fixed a problem where Solid Edge Standard Part fails to run.
PR 5559631: Fixed a problem where Solid Edge crashes when activating parts in this assembly.
PR 5519203: Fixed a problem with Parasolid import performance
PR 5560299: Fixed a problem where move triad does not display in exploded view
PR 5519203: Fixed a problem with Step file translation performance. Registry files were provided to enable changes.
PR 1541780: Fixed a problem with Round command behavior changed in V19. Radius not keeping last value entered

=====Part=====

PR 5562891: Fixed a problem where deleting a sick relationship causes Solid Edge to abort.
PR 5566826: Fixed a problem where a file crashes with thread textures on
PR 1520837: Fixed a problem where a flat pattern will not update.
PR 1535616: Fixed a problem where switching between material and gage tabs does retain previous selections.
PR 5373443: Fixed a problem where Dual monitors get multiple min/max/exit buttons on windows, using XP

=====Draft=====

PR 1516308: Fixed a problem where a mirrored block was exported to in the wrong orientation
PR 1536329: Fixed a problem with smart hole depth in hole table.
PR 2068224: Fixed a problem where DXF files can't be imported if Japanese characters are in the file/path name
PR 5395786: Fixed a problem where it is extremely slow saving DXF files to a mapped drive.
PR 1533939: Fixed a problem where SE doesn't import dimensions from file
PR 5569104: Fixed a problem where Several help .chm files are missing on 2D Drafting (free) CD.
PR 5560331: Fixed a problem where you can't save a drawing to IGES
PR 5586249: Fixed a problem where hatching that is not shown reappears when converting a view to 2D.

=====API=====

PR 5556450: Enabled the Migration commands for a Solid Edge Foundation License
PR 1540237: Fixed a problem with linking against latest version of SEPartReader.zip
PR 1529781: Fixed a problem where StructuralFrame.Get(Set)OverrideEndCondition does not work
PR 1507804: Fixed a problem adding hole features using the API.
PR 1507804: Fixed a problem with Adding hole features using the API.

=====Insight=====

PR 1534720: Fixed a problem where links can not be resolved after LCA has been used multiple times
PR 2069673: Fixed a problem where the title of a managed document (Insight) found in search would not display
PR 5534235: Fixed a problem where the viewer is not displaying files in a erp system
PR 5566266: Fixed a problem where Solid Edge locked up in attempt to IPA a managed baselined part doc
PR 5567882: Fixed a problem where you cannot place parts from advanced search in parts library
PR 1523670: Fixed a problem where LCA path could not point to a subsite
PR 2070812: Fixed a problem where Cache Manager list box isn't working
PR 5584610: Fixed a problem where Revision Manager does not list the links in this drawing.
PR 5586542: Fixed a problem where PCFs are blank on open.
PR 5589795: Fixed a problem with Difference from V17. Can not choose a site level for the destination
