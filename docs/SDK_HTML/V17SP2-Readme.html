Release Notes:
--------------

Product:        Solid Edge
Version:        17.00.02.04
Date:           27-Jun-2005

This release of Solid Edge has been certified to run on the following:
 • Windows 2000 Professional® operating system running Service Pack 4.
 • Windows XP Professional® operating system running Service Pack 1 or 2.
 • Internet Explorer 6.0 or higher


This Update should be installed on top of Solid Edge Version:
17.00.00.65, 17.00.01.06


--------------------
End of Release Notes
--------------------

------------------------------------
This Update addresses the following:
------------------------------------

=====Assembly=====

PR 5189682: Fixed a problem which addressed display related crashes captured via customer submitted crashlogs.
PR 1391151: XpresRoute relation automation has been enhanced to return Reference elements.
PR 5162897: Fixed a problem where assembly sketches disappear from a drawing view
PR 5167399: Fixed a problem in frames which was causing the frames not to update correctly.
PR 5183278: Fixed a problem with constraints not updating correctly after a replace between FOA members.
PR 1384574: Performance for opening large assemblies in Property Manager has been improved.
PR 1377181: Fixes a problem with Web Publisher not publishing properties for parts in an assembly.
PR 1403241: Fixed a problem in which you could not save the file after creating a CAM constraint.
PR 5162968: Fixes a problem with not being able to place simplified assemblies as simplified.
PR 5203598: Fixed a problem with crashes that could occur following Save As X_T or Reports as captured via crashlogs.
PR 1401263: Fixed a problem where Solid Edge crashes in VS+ after SaveAsImage
PR 5140867: Fixed a problem with subassemblies being incorrectly marked as adjustable.
PR 5161476: Fixed a problem with a display anomaly when using a separate window to place parts.
PR 5180530: Fixed a problem where saving a family member with the "save member as" was leaving extra links.
PR 5212021: Fixed a problem that resolved a performance issue when using graphics card driven advanced.
PR 5190058: Fixed a problem where a copied sketch was not maintaining its relative position after updating the assembly.
 Please Note: This fix requires that the parent part be activated in the assembly before making changes.
PR 5203211: Fixed a problem where physical properties was returning different results based on selection method.
 Please Note, V17 requires the entire ASM be selected for the correct mass to be calculated.


=====Part=====

PR 5195181: Fixed a problem where insert part copy would allow a cyclic dependency.
PR 1394695: Fixed a problem of not taking plane input for intersection curves api.
PR 1400009: Preventing crash inside math.
PR 1400861: Fixed a problem renaming multi-hole features. 
PR 5160913: Fixed a problem where numeric separator is changed after creating a part copy of a UG file.
PR 5183185: Fixed a problem where Up line and down line information in the output file is incorrect
PR 5197656: Fixed a problem that will now reduce memory consumption when working with extremely large part models.
PR 5155310: Fixed a case-specific abort when opening file
PR 1404346: Fixed a problem where Delete Holes by Operator was not working.
PR 5170921: Fixed a problem of disabling the trim surface and split surface buttons.
PR 5201694: Fixed a problem with file save failure after changing bead material.
PR 1398448: Fixed a problem that corrected an out of memory while copy/paste in profile segments.
PR 1406329: Fixed a problem that caused a crash because FOP names exceeded max length
PR 2050161: Fixed a problem where the units of mass moments of inertia cannot not be changed.
PR 5205166: Fixed a problem where License Borrowing will now work when the license file contains comments.
PR 5205636: Fixed a problem where you could not create a part copy of a JT file created in NX.
PR 5205928: Fixed a problem where trim was always trimming only on one side.
PR 5206402: Fixed a problem where Solid Edge now displays a more sensible error message when an out of memory condition occurs during file open.

=====Draft=====

PR 5176538: Fixed a problem where SE aborts when this V12 file is opened.
PR 5180284: Fixed a problem of exporting the given draft file to AutoCAD causing Solid Edge to abort
PR 5184685: Fixed a problem of aborting while exporting the given draft file to AutoCAD
PR 5196160: Fixed a problem where locating a textbox is leading to an abort.
PR 5197668: Fixed a problem that caused Solid Edge to abort in undo after using edge painter.
PR 1395118: Fixed a problem translating drawing views to DWG when the specified config is not found.
PR 1400365: Fixed a problem with hatching translated from a metric draft file
PR 1402891: Fixed a problem where multiple page hole tables drift apart on update.
PR 2048446: Fixed a problem associated with undoing the creation of the first line in the cutting plane environment.
PR 2049002: Fixed a problem where linear group dimensions on detail views start from the locate point
PR 5177125: Fixed a problem where stack angular group dimensions are not getting exported correctly.
PR 5177989: Fixed a problem where the callout text is not getting translated out correctly.
PR 5177992: Fixed a problem of fractional dimension display exporting to AutoCAD as decimal.
PR 5179023: Fixed a problem of hidden layers exporting visible during save as flat
PR 5188329: Fixed a problem where drawing views cannot be created after renaming "Normal" fill style.
PR 1402037: Fixed a problem where Solid Edge (v16, v17) would crash when opening some pre-v16 .DFT files.
PR 5183649: Fixed a problem that caused a crash in sheet setup.
PR 1399602: Fixed a problem where ModelLink.Range3d returned an error.
PR 1403923: Fixed a problem with mirroring groups and centerlines
PR 1406215: Fixes a problem where a parts list isn't displayed after being created.
PR 2049537: Fixed a problem where watermark is garbled in Solid Edge Japanese version.
PR 5195356: Fixed a problem that would cause Draft files to fail to open in V17.
PR 5184516: Fixed a problem where "exclude reference parts" causes sub-assembly's with no visible parts to be excluded from the Draft Parts List.

=====MoldTooling===== 

PR 5157786: Fixed a problem getting the correct number of sub-occurrences when queried from sub-occurrences API.
 
=====API=====
 
PR 1359270: Fixed a problem where OLE for DM container is not displaying SE Part in the embedded object site.

=====Insight=====

PR 1402916: Fixed a problem where SearchScope.txt entries were case sensitive.
PR 5177093: Fixed a problem where opening an assembly with Free Viewer write locks sub-assemblies.
PR 5185350: Fixed a problem where Insight Assistant fails on Jpn OS when WebParts page has JPN chars
PR 5190493: Fixed a problem where, Options file created by Insight Assistant on localized software won't update.
PR 5164307: Fixed a problem where the "Open in Dashboard" menu will now be shown if the user Right clicks on SharePoint URL in search results.
