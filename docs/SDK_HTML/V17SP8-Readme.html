Release Notes:
--------------

Product:        Solid Edge
Version:        17.00.08.03
Date:           06-Jan-2006


This release of Solid Edge has been certified to run on the following:
 • Windows 2000 Professional® operating system running Service Pack 4.
 • Windows XP Professional® operating system running Service Pack 1 or 2.
 • Internet Explorer 6.0 or higher


This Update should be installed on top of Solid Edge Version:
17.00.00.65, 17.00.01.06, 17.00.02.04, 17.00.03.04, 17.00.04.03, 17.00.05.03,
17.00.06.03, 17.00.07.03


--------------------
End of Release Notes
--------------------

------------------------------------
This Update addresses the following:
------------------------------------

=====Assembly=====

PR 2054713: Fixed a problem where in SE was crashing.
PR 2055300: Fixed a problem with deactivating an assembly with missing parts.
PR 2059763: Fixed a problem with RestoreAll/CollapseAll commands.
PR 2059883: Fixed a problem with Undo that caused assembly to crash.
PR 5355466: Fixed a problem that caused Place Part/Assemble to crash
PR 5365270: Fixed a problem to avoid a potential crash situations during a display event.
PR 5365704: Fixed a problem with CUnknownEnumDataHolder::SetData that caused a crash
PR 1470190: Fixed a problem with Assembly pattern not using incorrect matrix
PR 1471061: Fixed a problem where assembly patterns get deleted when switching to simplified assembly
PR 1476126: Fixed a problem where Assembly Feature (counterbore hole) is not updating.
PR 2055978: Fixes a problem with being able to locate an assembly reference plane in FlashFit
PR 2059760: Fixed a problem with windows not responding in the assembly environment.
PR 2059865: Fixed a problem with multiple parts being displayed when in-place active w/Hide Previous Level on.
PR 5348127: Fixed a problem with corrupt config files causing crashes in Draft.
PR 5357587: Fixed a problem with highlight in assembly.
PR 5357976: Fixed a problem to avoid potential corruption of current view camera.
PR 5365173: Fixed a problem where Iconnect and SE Viewer crash when exiting
PR 5365520: Fixed an assembly crash.
PR 1477908: Fixed a problem that caused a crash when changing end conditions to round in frames.
PR 2060266: Fixed a problem where some parts were activating when an assembly was opened
PR 5326912: Fixed a problem that caused an abort while applying assembly constraint
 
=====Part=====

PR 2059707: Fixed a problem that caused an Abort when dragging is coupled with view manipulations
PR 2059862: Fixed a problem where Direct Move was causing some commands to be disabled if the feature failed.
PR 2059863: Fixed a problem where Direct Rotate was not updating the angle correctly in the axis step.
PR 2059864: Fixed a problem where Direct Offset edit was not updating the direction.
PR 5353233: Fixed a problem that under defined property not implemented on the SKETCH API
PR 5354104: Fixed an abort in CLiteData::LoadBodies
PR 5357429: Fixed a problem in a function call to avoid stack corruption
PR 5358951: Fixed a problem to ensure we use lowercase for application ref field for PLMXML document.
PR 2060125: Fixed a problem where undoing during the edit of a loft caused the document to close.
PR 1447589: Fixed a problem that was causing a performance regression.
PR 1476638: Fixed a problem where an abort can occur due to changes in a bspline curve is fixed.
PR 2060416: Fixed a problem where duplicate icons appear on the menu bar when minimizing and maximizing
PR 5301382: Fixed a problem where in we were sometimes unable to load part copies of UG PRT files.
PR 5359160: Fixed a problem when a series of solves contains a failed solve.
PR 5359436: Fixed a problem where imported styles were incorrect when saved as to Parasolid

=====Draft=====

PR 1471118: Fixed a problem where Draft file aborts during file open
PR 1447933: Fixed a problem where retrieve dimension is not retrieving all the dimensions from the 2D Model sheet.
PR 5337009: Fixed a problem with conversion of thread data from old files.
PR 1478451: Fixed a problem where groups with projection lines are behaving strangely
PR 5353337: Fixed a problem with shaded drawing files of part and weldment files having the wrong color
PR 5362261: Fixed a problem with the fence locate of hidden cutting plane lines
PR 5365221: Fixed a problem with a memory overwrite in printing that may cause crashes.
PR 1401017: Fixed a problem where Group names will now be preserved on copy.
PR 4673160: Fixed a problem with selecting group elements in the stretch command
PR 5350928: Fixed a problem were a Label weld cross section is reset to zero.
PR 5366571: Fixed a problem with hatch display
 
=====API=====

PR 2059928: Enhancements have been made to the Application Display Automation controls to allow programmatic selection of Application Display settings.
PR 5368064: Fixed a problem where .NET addins fail to add their menus and toolbars.

=====Insight=====

PR 2055062: Fixed a problem where Web paged embedding SEPartX control can now be opened using hyperlinks
PR 1471917: Fixed a problem where connection to Library failed while going from LCA Step 1 to Step 2
PR 5193105: Fixed a problem that was not allowing a replacement of a NX2 file with a NX3 file in Revision Manager.
PR 5361319: Fixed a problem where the contents of a lib are sorted or grouped according to their type for view and selection.
PR 1472103: Fixed a problem where add to library does not fail when prevent duplicate filename option is on.
PR 5370924: Fixed a problem where ATL of a file with custom properties having no names.
PR 1468781: Fixed a problem where "#" will now be allowed as revision delimiter. Can use for non-Insight files only.
PR 5330720: Fixed a problem when saving a file with broken links.
PR 5371185: Fixed a problem where "Add To Library" fails due to a file whose links cannot be read.
PR 5347635: Fixed a problem where Property Manager does not list any FOP Children when in the master file and properties can not be modified for that member in case of managed files
PR 5364240: Fixed a problem where NX PRT files having 2D data in CGM format can be viewed properly.
