----------------------------------------------------------
This Update addresses the following Customer Reported PRs:
----------------------------------------------------------

=====Assembly=====

PR 5703545: Fixed a problem where occurrence overrides not always being set up correctly.
PR 2080435: Fixed a problem where switching members results in a message displayed about not having write-access to cfg file
PR 5726327: Fixed a problem where assembly pathfinder is not showing any subassemblies or parts.
PR 5737415: Fixed a problem where you do not get mimimum distance measurement results
PR 1589329: Fixed a problem where assembly features fail to recompute
PR 5600154: Fixed a problem where subassembly selection is not honored for cut items in a section.
PR 5737987: Fixed a problem where assembly features do now show up correctly
PR 1576343: Fixed a problem where assembly features are not updating when updating variables driving it.
PR 1589086: Fixed a problem where Occurrence Properties X,Y,Z fields no longer editable in Read-only
PR 5743069: Fixed a problem where assembly features was causing Solid Edge to crash on open.
PR 5731968: Fixed a problem where you were unable to create assembly arc pattern features.
PR 5742057: Fixed a problem where you were unable to edit a circular assembly pattern feature to be an arc feature.
PR 5747601: Added code to prevent one way of creation of loops in assembly documents with assembly features.

=====Part=====

PR 5734088: Added support for NX5 files in V19
PR 1585846: Fixed a problem where a tab with a contour flange flattens to a different size than a hem
PR 2077594: Fixed a problem where protrusion extent does not stay associative to keypoint.
PR 1588317: Fixed a problem where modifying a thread using 'edit definition'-step does not work
PR 5734088: Added support to read NX5 files in V19
PR 1592749: Fixed a problem where edge highlight is not accurate in large size parts.
PR 5746678: Fixed a problem where dimensioning between holes in 64bit V19 caused Solid Edge to crash.

=====Draft=====

PR 5681877: Fixed a problem where a 3D DWG file does not import SAT from ACAD 2004
PR 1579671: Fixed a problem where a V16 draft file cannot be opened in V19
PR 1564455: Fixed a problem where a Draft file aborts when opened
PR 5687761: Fixed a problem where PMI property text gives bad data w/simplified assemblies
PR 5676161: Fixed a problem where a painted edge is going away on DV update
PR 5733516: Fixed a problem where it is not possible to select some text elements by drawing a fence.
PR 5745631: Fixed a problem where edge display was incorrect.
PR 5738081: Fixed a problem where a drawing view would not update
PR 1564455: Fixed a problem where a Draft file aborts when opened

=====API=====

PR 1534463: Fixed a problem with the Insight.CopyFiles API
PR 1587824: Fixed a problem with Assembly Structure API performance

=====Insight=====

PR 5712946: Fixed a problem where Doc Number Property is being cleared in File Properties using Set action to revise.
PR 2080142: Fixed a problem where Japanese text is not displayed correctly in View & Markup
PR 1571569: Fixed a problem creating a revision from within Solid Edge
PR 1590305: Fixed a problem where Columns in Properties Manager are not shown in SE V19 64 bit version
PR 1590941: Fixed a problem where Revision Manager is updating the wrong links
