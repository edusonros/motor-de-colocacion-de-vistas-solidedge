Release Notes:

Product:        Solid Edge
Version:        18.00.03.04
Date:           17-Feb-2006


This release of Solid Edge has been certified to run on the following:
    • Windows XP Professional® operating system (32-bit) running Service Pack 2
    • Windows XP Professional® operating system (64-bit) running on EM64T hardware
    • Windows 2000 Professional® operating system running Service Pack 4
    • Internet Explorer 6.0 or higher


This Update can be instaled installed on top of Version: 18.00.00.69, 18.00.01.08,
or 18.00.02.04.


Note: Updates are all inclusive and can be installed on top of the base release
or any released UPdate.

This Update addresses the following:
------------------------------------

=====Assembly=====

PR 5366576: Fixed a problem where entries in Calculation Tab do not increment with Japanese version
PR 5370911: Fixed a problem where you cannot select patterned parts in pathfinder when exploding
PR 1494460: Fixed a problem where Peer variables are not working when assembly is opened with all inactive parts.
PR 2060582: Fixed a problem with delimitation line of add-in menu
PR 5336939: Fixed a problem where Assembly feature holes are losing relationships after an edit.
PR 5355277: Fixed a problem where Add buttons (Shaft dialog) are missing directional arrows on Jpn Eng Ref
PR 5361280: Fixed a problem where "edit in SE compression spring designer" doesn't work
PR 5391954: Fixed a problem where Asm is not showing part geometry when a variable in the top level is changed
PR 1476826: Fixed a problem where relationships are not updating after editing parts.
PR 1494338: Fixed a problem where you were unable to convert an assembly to STEP
PR 1494392: Fixed a problem with Standard Parts - English names in German version.
PR 5365219: Fixed a problem with typos in Russian Standard Parts UI
PR 5395519: Fixed a problem where the fillet weld options dialog not showing the correct values when editing a fillet weld.
PR 5384474: Fixed a problem where printing reports truncates column titles on Jpn version
PR 5399647: Fixed a problem with truncated text on a dialog for Extension Spring in Engineering Reference.
PR 5402912: Fixed a problem in Engineering Reference where the teeth in small spurgears will not get created.

=====Part=====

PR 1483470: Fixed a problem that caused when placing Fillet in Sketch with Included edges
PR 2060127: Fixed a problem with incorrect value of mass when 'Show units in value fields' is unchecked.
PR 5313893: Fixed a problem where embedded periods are allowed in FOP member names.
PR 5369357: Fixed a problem where Blue Dots fail to place when the point is a B-spline's control point.
PR 5374396: Fixed a problem where a Step file exported from SEv18 is at the wrong scale
PR 5391741: Fixed a problem where feet and inches do not show up correctly in parts list
PR 2062020: Fixed a problem where "Zoom to Error Elements" does not zoom to the correct elements when part/psm file is in place activated from asm.
PR 1466602: Fixed a problem where a Sheet metal part is not being computed when the material name is changed

=====Draft=====

PR 1451474: Fixed a problem where Drawing views do not trim correctly when exported to AutoCad format.
PR 5372853: Fixed a problem where this draft file fails to save after updating views.
PR 5371659: Fixed a problem where detached dimension can't be re-attached or replaced
PR 5384074: Fixed a problem where you cannot open this Draft file after saving
PR 1494225: Fixed a problem importing DWG into V18 - no problem with V17
PR 1500968: Fixed a problem where dimensions jump to detail views when converting a view to 2D.
PR 2060967: Fixed a problem where Detail View translation does not translate custom border
PR 5384067: Fixed a problem where parts list is not being set out of date after changing occurrence properties.
PR 5404583 : Fixed a problem where multiple 2d model views are created on dwg export paper space
PR 5390686: Fixed a problem where changing a dimension to dual units results in an abort.
PR 1493661: Fixed a problem where V15 user views with non-rectangular cropping become uncropped.
PR 1495867: Fixed a problem with Unable to open Draft File in V18.
PR 5373067: Fixed a problem with Draft views becoming corrupt
PR 5391158: Fixed a problem where a draft file will not open in V18
PR 5403694: Fixed a problem where Dimensions connected to bend centerlines are detaching on file open.
PR 5408896 : Fixed a problem where file size drastically increased when exporting to dwg paper space.

=====API=====

PR 2060032: Added two new API ApprenticeMode and ShowStartupScreen on the Application object
PR 1481190: Fixed a problem where API method CreateReference creates a reference to the wrong sketch
PR 1479222: Fixed a problem with API: SEPartX.OCX
PR 5329905: Added API ModelMember.ShowEdgesHiddenByOtherParts.

=====Insight=====

PR 5389021: Fixed a problem that caused an aborts when opening this file with Solid Edge or RevMan.

PR 1499355: Fixed a problem with Where Used not working
PR 5397191: Fixed a problem where you are Unable to Save or use Add to Library for a particular part file
PR 5401669: Fixed a problem where Solid Edge search is crashing the w3wp.exe (SharePoint) process.
PR 5404652: Fixed a problem revising a part
