Release Notes:
--------------

Product:        Solid Edge
Version:        16.00.02.04
Date:           03-Nov-2004

This release of Solid Edge has been certified to run on the following:
 • Windows 2000 Professional® operating system running Service Pack 4.
 • Windows XP Professional® operating system running Service Pack 1 or 2.
 • Internet Explorer 6.0 or higher


This Update should be installed on top of Solid Edge Version:
16.00.00.63,16.00.01.05

 

--------------------
End of Release Notes
--------------------

------------------------------------
This Update addresses the following:
------------------------------------

=====Assembly=====

PR 2041609: Fixed a problem with "As Last Saved".
PR 5041308: Fixed a problem with direction locking.
PR 5044298: Fixed a problem where reports do not work, when file is opened with hidden parts.
PR 5045733: Fixed a problem with Assembly Reports printing the first page only.
PR 5049368: Fixed a problem where a tube end treatment could not be changed to "none" through automation.
PR 5049732: Added support for printing 3D wireframe to PDF output devices.
PR 5047492: Fixed a problem where rendering images were skewing under certain conditions.
PR 5051007: Fixed a problem with performance when searching files on an unknown server.
PR 5054421: Fixed a problem with pattern occurrences being excluded in the Dynamic Configuration dialog.
PR 1317435: Fixed a problem with interference check on multiple body weldment files.
PR 1345388: Fixed a problem with missing holes in Assembly driven part features.
PR 4965891: Fixed a problem where you are unable to edit Assembly Revolved Cutout.
PR 5050230: Fixed a problem with From/To extents not working on assembly Hole feature
PR 1355479: Fixed an abort in Move Part.
PR 1354841: Fixed a problem with the performance of "Update Status" command.
PR 5014725: Fixed a problem with assembly features not working correctly when they were at multiple levels.
PR 5046182: Fixed a problem where a crash sometimes occurred when using the view command "Look at Face".
PR 5062652: Fixed a problem where BendTable automation does not return the correct values sometimes.
PR 5071955: Fixed a problem with losing styles and display settings while in-place active.
 
=====Part=====

PR 1350083: Fixed a problem with saved hole setting for Thread_Extent_Amount.
PR 1356188: Fixed a problem with split element causing an abort
PR 5045376: Fixed a problem with alignment and scale when using the Print Area command in Profile.
PR 1360182: Fixed a problem with loft which was causing a failure in some cases
PR 5050770: Fixed a problem where flat pattern fails on contour flange with close 3-bend corner
PR 5055723: Fixed a problem with Solid Edge Virtual Studio + not displaying correctly in Help About.
PR 5057291: Fixed a problem where Density variable values were not properly carried forward to FOP member files.
PR 1353213: Fixed a crash when placing/editing revolved features in certain files.
PR 5057688: Fixed a crash that occurred when using a custom bend equation for sheet metal documents.
PR 1359709: The problem where the MRU file list can be corrupted after opening a file from explorer is fixed.
PR 1360177: Fixed a problem  with the close corner command
PR 5043282: Readout for custom variable properties is now handled through File Properties->Advanced Units tab.
PR 5049430: Fixed a problem where feature libraries will now be created with he units of the source document.
PR 5070560: Fixed a problem where a pattern can become degenerate

=====Draft=====

PR 5017920: Fixed a problem where SE aborts when opening exported paper space DWG files.
PR 1348259: Fixed a problem where dimension text using a driven color of black is not output to AutoCAD.
PR 1354001: Fixed a problem with not being able to change the hatch style associated with a fill style.
PR 1356112: Fixed a problem with arrowheads shifting on import
PR 5026118: Fixed a problem where Constant attributes are not importing from DWG files.
PR 5043981: A watermark is now displayed on background sheets to indicate that a sheet is a background sheet.
PR 5045251: Fixed a problem in which dimensions would inexplicably detach from drawing views
PR 5045373: Fixed a problem with not being able to scroll with the mouse wheel.
PR 5052129: Fixed a problem copying and pasting drawing views into other applications.
PR 1362697: Fixed a problem importing a specific DXF file.
PR 2042263: Fixed a problem with centermark display
PR 5059990: Fixed a problem where parts lists are corrupted after copy/paste.
PR 5061215: A new option has been added to allow saving viewer data as black and white.
PR 5069164: Fixed a problem with section views and assembly feature display.
PR 5069642: Fixed a problem where a V12 dimension is changing value when opened in V15 or later.
 
=====API=====

PR 1359594: Fixed a problem with BSplineSurfaces.Add() method
PR 1355124: Fixed a problem where the correct number of faces is now returned for all direction vectors.
PR 5061319: Fixed a problem where GetDirectoryName API does not work with Insight
 
=====Insight=====

PR 1355331: Fixed a problem where; no data synchronization in SharePoint was taking place.
PR 5036316: Fixed a problem where saving a managed file in case of URL containing '//' would fail.
PR 5050060: Fixed a problem where the Insight Fixup utility did not work for SharePoint V2.
PR 5010194: Fixed a problem where similarly named doc libs were causing searchscope.txt to be used incorrectly.
PR 5059148: Fixed a problem where the status was being set to Available after a Rename action.
PR 5061902: Fixed a problem where managed documents open slow if cache needs to be synchronized.
PR 5069299: The check for Duplicate DocIDs during file migration can now be controlled by a registry switch SKIP_DUPLICATEDOCID_CHECK_DURING_SPMIGRATION.
 
=====MoldTooling=====
 
PR 5056349: Adds the Meusberger files.
PR 5043066: assorted fixes from the vendor.
