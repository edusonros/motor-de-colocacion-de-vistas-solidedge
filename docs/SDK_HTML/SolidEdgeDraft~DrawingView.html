Solid Edge Draft Type Library : DrawingView Object
Description
A 2-D representation of a 3-D part or assembly model.
Object Model
Remarks
A drawing view is used to display design space geometry in document space. A view of design space is enclosed by the drawing view border (a handle that allows manipulation of the drawing view). Only one part or assembly document can be used as the basis for drawing views in a draft document.
