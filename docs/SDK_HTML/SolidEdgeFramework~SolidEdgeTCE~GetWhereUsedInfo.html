pvarListFilesWithMFKAttributes
Returns the Two dimensional array of Where-Used results which includes Filename, Revision ID, Item type, Array of MFKAttributes.
Remarks
Two dimensional array structure is fixed, zeroth column always represent Dataset filename and so on.
Example:
ListFilesWithMFKAttributes [0,0] =  Dataset File name
ListFilesWithMFKAttributes [0,1] =  Item Revision
ListFilesWithMFKAttributes [0,2] =  Item Type
ListFilesWithMFKAttributes [0,3] =  Array Of MFK attributes[Key, Value]
Note: This API also support to get where-used documents info, If part and draft under same ItemRevision and there is no link in between them.
