    • Windows XP Professional® operating system (32 bit or 64 bit)
    • Windows Vista Business® or Vista Enterprise® operating system (32 bit or 64 bit)
    • Internet Explorer 7.0 (IE 6.0 minimum)

----------------------------------------------------------
This Update addresses the following Customer Reported PRs:
----------------------------------------------------------

=====Assembly=====

PR 5737340: Fixed a problem that caused a circular dependency error using FlashFit.
PR 1649779: Fixed a problem where the DELETE key in assembly does not work at times.
PR 2076104: Fixed a problem where in the user is unable to auto-restore a backed up file.
PR 1628066: Fixed a problem with FOA and Assembly Features
PR 2088091: Fixed a problem where it takes too much time and memory to update the "Physical Properties”
PR 2085635: Fixed an abort from data submitted via a crashlog.
PR 6022024: Fixed a problem with include that was introduced in Update 9.
PR 1681255: Fixed a problem where Save As Image does not create a rendered image in Foundation
PR 1586883: Fixed a problem where Part colors are not exported to step.
PR 6023710: Fixed a problem where washers are not available with C'snk screw
PR 6001319: Fixed a problem where embedded weldment information gets lost
PR 6026967: Fixed a problem where previous custom part inherits length of the most recently placed part.
PR 6033504: Fixed a problem where a revolved cutout is removing material where there is no sketch.
PR 6000242: Fixed a problem where you are unable to import an STL file.

=====Part=====

PR 6006751: Fixed a problem where the Hole Table is not listing the correct hole diameters for a counterbore
PR 1641087: Fixed a problem that was causing a crash with an old part in V20 SP3
PR 1668197: Fixed a problem where DIN.PZL will not be used for lofted flanges
PR 1680457: Fixed a problem where typing "=" and selecting dim adds multiple entries to variable table.
PR 5782978: Fixed a problem where moving the cursor over Variable table causes zoom to happen or automatically
PR 2105606: Fixed a problem where a Round disappears when part is recomputed in Version 20
PR 2102531: Fixed a problem where creating an open profile protrusion on a tapered face failed in V20
PR 2096263: Fixed a problem where you are unable to make a dimple that uses an inside edge.
PR 1603406: Fixed a problem with mirror performance on grouped 2D elements, maintain relationships on
PR 6040150: Fixed a problem with Tool Tips not displaying properly when using SmarTeam

=====Draft=====

PR 5992242: Fixed a problem where EMF's are scaled incorrectly
PR 5941495: Fixed a problem where a specific workflow with Blocks will cause a draft file to be unusable.
PR 5969985: Fixed an abort from data submitted via a crashlog.
PR 2107915: Fixed a problem where data alignment in table properties is incorrect
PR 6024647: Fixed a problem that caused Solid Edge to crash when copying a fill style
PR 2105954: Fixed a problem where a Japanese font is not mapped to an ACAD font.
PR 1684163: Fixed a problem where you cannot copy and paste a view that contains a coordinate dim
PR 1535069: Fixed a problem where you are unable to hide an overridden edge segment
PR 6031576: Fixed a problem where Solid Edge runs out of memory when working with shading Drawing views.
PR 1644606: Fixed a problem where a drawing view is never up to date
PR 1632537: Fixed a problem where you save a Solid Edge draft file as a pdf file and the output is rotated.
PR 6020990: Fixed a problem where you cannot import a draft file

=====API=====
 
PR 5544696: Fixed a problem with Expose Parasolid id/tag via API for Body, Edge, Face, Loop, Shell and Vertex
PR 1619947: Fixed a problem with a call to SurfaceBoundaries.Add failing with 'type mismatch'.

=====Insight/=====
 
PR 5549874: Fixed a problem where Parts are shown as Designed even though they are Simplified parts
PR 6029411: Fixed a problem where Solid Edge crashed when IPA on a part or assembly from a checked out Draft file.
